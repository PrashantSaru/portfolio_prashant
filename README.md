# Prashant Saru - Portfolio Website

A responsive personal portfolio website built as part of my Week 3 Web Development assignment (CSE230, International American University, Kathmandu).

**Live site:** https://prashantsaru.github.io/portfolio_prashant/

## About

I'm a BIT student working toward a career as a SOC analyst in cybersecurity, and I also do freelance web development on the side. This site includes a short bio, my skills, a couple of projects I've worked on, and a contact form.

## Built With

- HTML5 (semantic markup)
- CSS3 (Flexbox and Grid, no frameworks)
- Responsive design with media queries for tablet and mobile
- Hosted with GitHub Pages

## Features

- Clean separation of structure (HTML) and styling (CSS) — no inline styles
- Two-column responsive skills grid that collapses to one column on mobile
- Flexbox layout for the About section
- Mobile-friendly navigation and contact form

## Running Locally

Clone the repo and open `index.html` in your browser, or use the VS Code Live Server extension for live reload during editing:

```bash
git clone https://github.com/PrashantSaru/portfolio_prashant.git
cd portfolio_prashant
```
